
# EquipRentals - Equipment Renting System (MERN Stack)

**EquipRentals** is the premier online platform for renting a wide range of equipment and devices. Whether you need the latest tech gadgets, outdoor gear, or specialized tools for events, our platform offers a convenient, affordable, and hassle-free rental solution.

## 🌟 Key Features

- Browse and rent various types of equipment
- Secure user authentication and authorization
- Admin dashboard for managing users, products, and orders
- Payment options: Cash, Credit, and Online UPI
- Responsive UI for both desktop and mobile
- Promotes sustainability through short-term usage

## 💡 Mission

> **"Experience More, Own Less."**

EquipRentals is built with the idea of promoting sustainability and flexibility by allowing users to enjoy top-tier equipment without the burden of ownership.

## 🛠️ Tech Stack

- **Frontend**: React.js, Bootstrap
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT
- **Payment Integration**: Cash, Credit, UPI
